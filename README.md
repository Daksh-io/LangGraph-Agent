# 🚀 LangGraph AI Agent Chatbot 🤖🌐.

Welcome to the LangGraph AI Agent project! This solution enables you to interact with powerful AI models (Groq & OpenAI) with web search capability 🌐. Create smart, responsive AI agents that answer your questions, enhanced with external search functionality 🧠🔍.

## 🌟 Project Overview.

This project combines advanced AI language models with an easy-to-use UI and backend API system to deliver interactive chatbot experiences. The agent is powered by LangChain, Tavily Search, and OpenAI/Groq models, making it a flexible solution for data scientists, developers, or AI enthusiasts 🎯.

### ✅ Main Features.
- ⚡ Multiple Model Providers: Supports both GROQ and OpenAI models.
- 🌐 Optional Web Search: Extend the model’s knowledge by enabling web search.
- 🧱 Simple UI with Streamlit: User-friendly frontend to interact with your AI agent.
- 🚀 FastAPI Backend: Handles requests, validates input, and dynamically invokes AI models.
- 🔧 Configurable System Prompt: Customize the system behavior by defining your own prompt.
- 📚 Supports Multiple Models:
   - OpenAI Models:
     ```gpt-4o-mini```
   - GROQ Models:
     ```llama-3.3-70b-versatile```, ```deepseek-r1-distill-llama-70b```, ```qwen/qwen3-32b```

## 🎯 Example Usage.
1. Define your system prompt (e.g., "Act as a helpful data science assistant").
2. Choose the model provider (GROQ or OpenAI).
3. Select the specific model.
4. Optionally enable web search 🌐.
5. Enter your query 🤔.
6. Click Ask Agent! 🤖 to get instant responses 🎉.

## 🚨 Notes.
- Make sure your ```.env``` contains valid API keys to avoid authentication errors.
- The system supports dynamic model selection, allowing flexibility in experimentation 🔧.
- Ideal for prototyping AI chatbot solutions or integrating external search capabilities.
